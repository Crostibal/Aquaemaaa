¿Qué es aquaponia

A aquaponia consiste em um sistema de criação de peixes junto de um sistema de cultivação de plantas tendo a recirculação de água, nesse sistema os peixes produzem matéria orgânica que contribui para as plantas e as plantas produzem nutrientes que contribuem pros peixes, fazendo com que ambos se beneficiem.

Hardware

Materiais: - Válvulas solenoides - Arduino uno - Sensores de nível da água - Bombas de água - Relé A automação do sistema de aquaponia foi feita a partir de um arduino UNO para controle do sistema

Como funciona

- O relé foi utilizado para controle das válvulas solenoides e da bomba que controlam o fluxo da água - As válvulas solenoides foram utilizadas para controlar a entrada e saída de água nos canos do sistema aquaponico - As bombas da água foram utilizadas para distribuir a água pelo projeto, sendo uma usada para mandar água do reservatório para o tanque, e outra do tanque para a plantação - Os sensores de nível da água servem para verificarmos quando o sistema está em estado crítico e deve receber água do reservatório
